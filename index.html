Display and positioning

Property Description Values Example
Z-Index This property specifies how far back or how far foward an element will apear on a web page when it overlaps with other elements. Integer values, which can be negative or positive values. The highest will be at the foreground and the lowest will be at the back. .element {
position: absolute;
z-index: -1;
}
Position Specifies options for positioning HTML elements on a web page. Can be set to static, relative, absolute or fixed. Fixed - When the CSS position has a value of fixed, it is set/pinned to a specific spot on a page. The fixed element stays the same regardless of scrolling. div.fixed {
position: fixed;
bottom: 0;
right: 0;
width: 300px;
border: 3px solid #73AD21;
}
Absolute – Enables an element to ignore sibiling elements and instead be positioned relative to its closest parent element that is positioned with relative or absolute. div.absolute {
position: absolute;
top: 80px;
right: 0;
width: 200px;
height: 100px;
border: 3px solid #73AD21;
}
Relative – An element with relative position is positioned relative to where it would have originally been on a web page. The offset properties can be used to determine the actual position of the elemento relative to its original position. Without the offset properties the declaration will have no effect on the positioning. div.relative {
position: relative;
left: 30px;
border: 3px solid #73AD21;
}
Static - is the default positioning of HTML elements. div.relative {
div.static {
position: static;
border: 3px solid #73AD21;
}
Float Determines how far left or how far right an element should float within its parente element. The value left floats an element to the left side of its container and the value right floats na elemento to the right side of its conainer. For the property float, the width of the container must be specified or the element will assume the full width of its containing element. img {
float: right;
}
Clear Specifies how an elemento should behave when it bumps into another element within the same containing element. The clear is usually used in combination with elements having CSS float property. This determines which sides floating elements are allowed to float. None, left, right, both, inherit div1 {
float: left;
}

div2 {
clear: left;
}
